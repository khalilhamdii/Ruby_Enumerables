## Getting started

Before getting started make sure that ruby is installed.

- Start by cloning this repository to your local machine :

```bash
git clone git@github.com:khalilhamdii/Ruby_Enumerables.git
```

- Access the cloned repoository :

```bash
cd Ruby_Enumerables
```

- Run this command :

```bash
ruby main.rb
```
